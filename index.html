재료 매칭 확인 후 일괄 입고

데이터 백업 / 이전

지금 서버(Supabase)에 있는 데이터를 파일로 저장합니다. "예전 백업 파일 가져오기"는 이전 로컬/localStorage 버전에서 받아둔 백업 파일을 딱 한 번 서버로 옮길 때만 쓰세요 — 누를 때마다 새로 추가되니 반복 실행하지 마세요.

계정 안내

비밀번호 변경은 Supabase 대시보드 → Authentication → Users에서 admin@somri.local / staff@somri.local 계정을 선택해 직접 바꿔주세요. 이 앱 안에서는 더 이상 비밀번호를 관리하지 않습니다.
